athenaValue + HPPN Dashboards

⇦ Previous ⇨ Next

Problem

What is the future of the dashboard zone? HPPN, athenaValue and the “reporting home” are concepts we drew up to communicate future state of analytics

Solution

Mockup work, moving from a “home” into a more specific dashboard curated by a theme (in this case, athenaValue/ROI)

Ran an ROI Sketching Session (July 6, 2017), working with a product performance expert to help us brainstorm how an ROI dashboard could take shape and fit in with current reporting tools.

Built concepts (from wireframes to high fidelity mockups) for the ROI dashboard, displaying athena and practice-specific task breakdown

High Performance Physician Network (HPPN) metrics are selected from high-performing practices and used to improve practice performance for all clients. We laid out this information in a scorecard, and mocked it up at a metric-detail level.

Outcome

This dashboard work helps inform our vision for a “Dashboard Workbench”—components necessary for all dashboarding and data visualization needs across athena. All work lives in HPPN and ROI folders within the Analytics Dashboard Zone directory
